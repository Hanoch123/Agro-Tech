Farmer's Friend

This is a web based project which is useful for farmers and agricultural students. This is an open discussion portal providing solutions to farmers and agricultural students. Any general public can also use this system for knowing the information about various crops, and the usage of fertilizers.It provides all the users to have userID and password.Users can also post their queries.The main objective of the project is to provide information to the farmers and agricultural students.It provides better solutions for farmers in providing the information regarding soils,fertilizers.It also helps the agricultural students to get practical information regarding various crops.This portal also provides the recommended dose of NPK for various types of crops.
